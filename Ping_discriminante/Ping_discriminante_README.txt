*** INTRODUZIONE ***
Il seguente tool permette, data una lista di IP da monitorare da inserire nel file "iptest.txt".
Il tool crea un report in cui inserisce nel file "online.txt" gli ip raggiungibili, e nel file "offline.txt" quelli non raggiungibili, gli ip verranno scritti uno per riga.
Mostra inoltre a video durante la sua esecuzione in tempo reale quelli non raggiungibili.

-->	LANCIARE il bat "main.bat"
